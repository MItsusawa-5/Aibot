# アプリ名「Aibot」
まるで自分の相棒のようなLINEBotです。

## Description
基本的なLINEbotです。文章の返答や、時間計測などが出来ます。

## 今後実装予定
決まった時間にBotを飛ばしてくれる機能
返信しなかった場合、繰り返しBotを飛ばしてくれる機能

## 開発経緯
開発者自身の朝の弱さが発端。予定がないとすぐだらけてしまう。でも、以外とそういう人は多いかも？
LINEは私たちの生活に根付いたツールなので、時間を管理する時にこの上ない相棒になるはずです。

## Features

- python記法を用いた開発
- LINEBOTのSDKを用いた開発
- flaskを用いた通信
- ngrokを用いた環境で動作テスト

## Requirement

- python 3.6.5
